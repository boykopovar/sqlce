Просмотр SQL Server Compact (.sdf)

Откройте файл .sdf в браузере - без установки программ и без отправки файла на сервер.

Ограничений по размеру нет, но в текущей реализации файл попадает целиком в оперативную память. Так что если файл уж слишком большой, браузер может зависнуть.
